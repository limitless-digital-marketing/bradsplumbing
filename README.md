# McKays Plumbing and Gasfitting Website

A modern, conversion-focused website for McKays Plumbing and Gasfitting.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

- **Core Pages**: Homepage, Contact, Quote, Blog, Main Service Pages
- **SEO Pages**: Service and location-based pages using reusable templates
